Vacuum Cleaner Robot

A robotic vacuum cleaner, sometimes called a robovac or a Roomba as a generic trademark, is an autonomous robotic vacuum cleaner which has a limited vacuum floor cleaning system combined with sensors and robotic drives with programmable controllers and cleaning routines. Early designs included manual operation via remote control and a "self-drive" mode which allowed the machine to clean autonomously.

“Design and Manufacturing of Automatic Switching Control in Hybrid Vehicle System for two-wheeler”

This project focuses on designing and manufacturing an automatic switching control system for a hybrid two-wheeler. The system aims to optimize fuel consumption and performance by seamlessly switching between the electric motor and internal combustion engine based on real-time factors like speed, load, and battery charge. This is achieved by developing a sophisticated control algorithm and integrating it with the vehicle's powertrain components.
